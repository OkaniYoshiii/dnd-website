# Site internet D&D (Donjons et Dragons)

Site internet permettant de créer une fiche de personnage pour le jeu "Donjons et Dragons" mais aussi de pouvoir modifier ces statistiques en cours de partie.

## Prérequis

Avoir installé les utilitaires suivants :

- Nodejs

## Stack technique

- Typescript

## Installation

Installation des dépendances : `npm install`
Lancement du projet en mode "dev" : `npm run dev`
